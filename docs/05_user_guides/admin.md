# Admin User Guide

## Overview
The Admin Dashboard is the central command center for District Directors and System Administrators. It allows for high-level management of councils, members, and organization-wide settings.

## Getting Started
**Login:** Access the login page and enter your Admin credentials.
**Dashboard:** Upon login, you will see the Admin Dashboard (`admin_dashboard.html`), displaying:
- Total Users (Active/Pending)
- Member Distribution Charts
- Daily Bible Verse
- Upcoming Global Events

## Core Features

### 1. Council Management
**Navigation:** `Management > Manage Councils`
- **Add Council**: Click `+ Add Council`, enter the Council Name, District, and Location.
- **Edit/Delete**: Use the action buttons next to each council in the list.
- **Analytics**: View specific analytics for each council.

### 2. User Management
**Navigation:** `Management > Members`
- **Pending Approvals**: Go to `Pending Users` to review sign-up requests.
  - **Approve**: Assigns the user to their selected council with the role 'Member'.
  - **Reject**: Deletes the request.
- **Role Management**: Go to `Role Management` to promote users (e.g., Member -> Officer) or demote them.
- **Archiving**: Soft-delete users who are no longer active via the `Archive` button.

### 3. Event Approval
**Navigation:** `Events > Proposals`
- Review event proposals submitted by Officers.
- Check event details, budget, and location.
- **Approve**: Publishes the event to the calendar.
- **Reject**: Sends the proposal back with a rejection reason.

### 4. Global Analytics
**Navigation:** `Analytics > Global Overview`
- View donation trends across all councils.
- Analyze recruitment metrics (which councils are growing fastest).
- Export reports as PDF or CSV.

### 5. Blockchain Ledger
**Navigation:** `Donations > Blockchain Ledger`
- View the immutable record of all transactions.
- Download the full ledger for auditing.
